
File descriptions to run CHARMM

The file names are given according to format
PDBid_OPEN/CLOSE/OCCLUDED_IonicStrength_EQ.CRD
PDBid_OPEN/CLOSE/OCCLUDED_IonicStrength_EQ.RST

EQ stands for equilibrared coorinates or restart files
.CRD is coordinate file
.RST is restart file
.inp files are CHARMM input files (using the same file name as CRD and RST)
dhfr.seq is the DHFR sequence file

6cw7, 1ra2 and 3ql3 are the PDBids of OCCLUDED, OPEN and CLOSED 
representative structures
